# Vatos Studio - Proyecto Web

## Descripción
Tienda/studio web llamada **VATOS STUDIO**. Sitio estático con HTML, CSS y JavaScript vanilla.

## Estructura
- `index.html` — página principal
- `styles.css` — estilos globales
- `main.js` — lógica principal (carrito, UI, interacciones)
- `CSS/` — estilos adicionales
- `JS/` — scripts adicionales
- `img/` — imágenes y assets

## Stack
- HTML5, CSS3, JavaScript vanilla (sin frameworks)
- Sin bundler ni build step — los cambios en archivos son directamente el resultado final

## Flujo de trabajo
- Los cambios se hacen directamente en los archivos y se pushean a GitHub (`main` branch)
- Para ver los cambios en VS Code: `git pull origin main`
